Создать: позитивные тест-кейсы на логотип

Тест-кейс 1: Проверка видимости логотипа
1. Открыть любую страницу сайта.
2. Обратить внимание на логотип в верхнем левом углу.
Ожидаемый результат:
1. Страница сайта открывается успешно.
2. Логотип четко виден и отражает идентичность бренда.

Тест-кейс 2: Проверка кликабельности логотипа
1. Открыть любую страницу сайта, кроме главной.
2. Кликнуть по логотипу в верхнем левом углу.
Ожидаемый результат:
1. Страница сайта открывается успешно.
2. После клика по логотипу происходит переход на главную страницу сайта.

Тест-кейс 3: Проверка согласованности логотипа
1. Открыть любую страницу сайта.
2. Сравнить логотип с цветовой схемой и стилем сайта.
Ожидаемый результат:
1. Страница сайта открывается успешно.
2. Логотип согласован с цветовой схемой и стилем сайта и не выделяется на фоне.

Создать: позитивные тест-кейсы на Каталог

Название кейса: Проверка отображения категорий товаров в каталоге
Шаги кейса:
1. Открыть страницу Каталога
2. Просмотреть список категорий товаров в левой части экрана
Ожидаемый результат:
1. Страница Каталога открывается успешно
2. В списке категорий товаров присутствуют все категории, представленные на сайте

Название кейса: Проверка поиска товара по названию в каталоге
Шаги кейса:
1. Открыть страницу Каталога
2. Ввести название искомого товара в поле поиска в верхней части экрана
3. Нажать кнопку Поиск или клавишу Enter
Ожидаемый результат:
1. Страница Каталога открывается успешно
2. Поле поиска принимает введенное значение
3. На странице Каталога отображаются товары, соответствующие введенному названию

Название кейса: Проверка фильтрации товаров по цене в каталоге
Шаги кейса:
1. Открыть страницу Каталога
2. Выбрать интересующую категорию товаров из списка в левой части экрана
3. Установить желаемый диапазон цен с помощью ползунков в правой части экрана
4. Нажать кнопку Применить фильтры
Ожидаемый результат:
1. Страница Каталога открывается успешно
2. Выбранная категория товаров активируется в списке и отображается в заголовке страницы Каталога
3. Ползунки цены перемещаются в соответствии с установленным диапазоном
4. На странице Каталога отображаются только товары, цена которых находится в заданном диапазоне

Название кейса: Проверка сортировки товаров по популярности в каталоге
Шаги кейса:
1. Открыть страницу Каталога
2. Выбрать интересующую категорию товаров из списка в левой части экрана
3. Выбрать опцию Сортировать по популярности из выпадающего списка в правой части экрана
Ожидаемый результат:
1. Страница Каталога открывается успешно
2. Выбранная категория товаров активируется в списке и отображается в заголовке страницы Каталога
3. Опция Сортировать по популярности выбирается в выпадающем списке и отображается рядом с ним
4. На странице Каталога товары располагаются в порядке убывания популярности (количество отзывов, рейтинг, продажи и т.д.)
